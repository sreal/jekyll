Windows rsync

Need to rsync in windows?

  1. Get `cwRsync` https://www.itefix.net/content/cwrsync-free-edition
  2. c:/applications/cwRsync/bin/> ./rsync.exe -e './ssh' -Pazhrvv <REMOTEUSER>@172.20.11.50:/internet/web/ ./<LOCALFOLDER>/
  3. Using the included ssh.exe (`-e ;./ssh`) avoid problems from not having the exact cgwin ssh version it expect installed.
  4. Cert may fail, <remoteuser> will prompt for pass (if setup)/
