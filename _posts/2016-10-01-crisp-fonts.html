Crispy Fonts

When importing fonts in Unity they will appear blurry at larger sizes.

You need to create a copy to edit the Filter mode.

Make a copy of the font

  1. Import the font to the into Resources folder
  2. In the Font's Inspector, change Rendering Mode to Hinted Raster
  3. In the Font's Inspector, change Character to ACSII default set
  4. In Settings (the cog), Create Editable Copy.
  5. In the created Import Settings, change Filter Mode to Point (no filter), Apply

Problem with this is it will not respect setting colors.
